\doxysubsection{Detailed Description}
This structure hold contact results for link pairs. 

A custom class was implemented to avoid a large number of heap allocations during motion which avoids full clearing the map. This class provides methods const container methods for access the internal unordered\+\_\+map and has two distinct different when it comes to the clear, size and release methods.

The clear method does not call clear on the unordered\+\_\+map but instead it loops over all entries and calls clear on the vector being stored. This allows the memory to remain with the map and not get release for each of the vectors stored in the map.

The size method loops over the map and counts those that have vectors which are not empty.

The release method actually calls clear on the internal unordered\+\_\+map relasing all memory.
